# Azure Core libraries for C++

Azure Core libraries provide shared functionality for Azure C++ client libraries.

## Libraries

|Reference|Package|Description|Source|
|---|---|---|---|
|[Core](core-readme.md)|[azure-core-cpp](https://vcpkg.io/en/package/azure-core-cpp)|Provides shared primitives, abstractions, and helpers for modern C++ Azure SDK client libraries.|[GitHub](https://github.com/Azure/azure-sdk-for-cpp/tree/main/sdk/core/azure-core)|
|[AMQP](core-amqp-readme.md)|[azure-core-amqp-cpp](https://vcpkg.io/en/package/azure-core-amqp-cpp)|Provides functionality for Azure C++ client libraries that use AMQP protocol.|[GitHub](https://github.com/Azure/azure-sdk-for-cpp/tree/main/sdk/core/azure-core-amqp)|
|[Tracing](core-tracing-readme.md)|[azure-core-tracing-opentelemetry-cpp](https://vcpkg.io/en/package/azure-core-tracing-opentelemetry-cpp)|Provides functionality for Azure SDK clients to emit telemetry using the OpenTelemetry API.|[GitHub](https://github.com/Azure/azure-sdk-for-cpp/tree/main/sdk/core/azure-core-tracing-opentelemetry)|
